Функция print()

Функция print() - отдельно выделенный алгоритм, названный по имени,
необходим для вывода информации, помещённой в качестве аргументов,
на экран пользователя.

Функция input()

Функция input() - отдельно выделенный алгоритм, названный по имени,
необходима для ввода информации, от пользователя. И вывода подсказки,
помещённой в качестве аргумента.

Функция len()

Функция len() - отдельно выделенный алгоритм, названный по имени,
необходима для подсчёта количества символов в строке.

Функция range()

Функция range() - отдельно выделенный алгоритм, названный по имени,
необходима дполучения последовательности в диапазоне
